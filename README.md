# Portfolio

A modern, minimalist portfolio showcasing projects and skills.

## Live Demo

[View Live Portfolio](https://gokul-codes-420.github.io/portfolio-main/)

## Features

- **Minimalist Design**: A clean and professional look with refined typography.
- **Project Showcase**: Vertical list of featured projects with detailed banners.
- **Glassmorphism UI**: Modern UI elements with subtle animations.
- **Responsive Layout**: Works seamlessly across devices.

## Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/gokul-codes-420/portfolio-main.git
   ```
2. Open `index.html` in your browser.
